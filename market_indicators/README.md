# 市场综合预警仪表盘 (Market Indicators Dashboard)

## 概述

这个工具计算并展示 5 个关键的市场预警指标，帮助你在复杂的市场环境下做出更理性的投资决策。

## 指标说明

### 1. ERP (股权风险溢价) - Equity Risk Premium
- **公式**: 股票盈利收益率 (1/PE) - 10年期国债收益率
- **预警线**: < 3.0%
- **含义**: 衡量持有股票相对于债券的"性价比"

### 2. 巴菲特指标 (Buffett Indicator)
- **公式**: 总市值 / GDP
- **预警线**: > 180%
- **含义**: 衡量虚拟经济相对于实体经济的过热程度

### 3. 高收益债利差 (HY Credit Spreads / OAS)
- **指标**: 美银美林高收益债期权调整利差
- **预警线**: > 500 bps
- **含义**: 债务危机的"金丝雀"，衡量企业的生存压力

### 4. 净流动性指标 (Net Liquidity)
- **公式**: 美联储资产负债表 - 财政部账户余额(TGA) - 逆回购(RRP)
- **预警线**: 趋势判断
- **含义**: 市场上的钱是在增加还是减少

### 5. 20法则 (Rule of 20)
- **公式**: 标普500 P/E + 通胀率
- **预警线**: > 20
- **含义**: 快速判断市场是否过热

## 安装

```bash
pip install -r requirements.txt
```

## 获取 FRED API Key (推荐)

为了获取最准确的数据，建议注册免费的 FRED API Key:

1. 访问 https://fred.stlouisfed.org/docs/api/api_key.html
2. 点击 "Request API Key"
3. 注册并获取你的 API Key
4. 设置环境变量:
   - Windows: `set FRED_API_KEY=your_api_key`
   - Linux/Mac: `export FRED_API_KEY='your_api_key'`

## 使用

```bash
# 基本使用 (会使用估计值)
python market_indicators.py

# 使用FRED API获取最新数据 (推荐)
set FRED_API_KEY=your_api_key
python market_indicators.py
```
